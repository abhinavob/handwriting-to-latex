# Handwritten Math to LaTeX

This project aims to convert handwritten math images into LaTeX.

Idea:
- take an image of handwritten equations
- process it using a deep learning model
- generate the corresponding LaTeX code

## Dataset
Using HME100K (not included in repo due to size)  
Download from Kaggle: https://www.kaggle.com/datasets/cutedeadu/hme100k
